# Code of Conduct

## Our Standards

PromptG is an open, inclusive community. We expect all participants to:

**Be respectful**

- Treat everyone with courtesy and professionalism
- Welcome newcomers and help them learn
- Assume good intent in discussions

**Be constructive**

- Focus on ideas, not individuals
- Provide actionable feedback
- Accept criticism gracefully

**Be collaborative**

- Share knowledge openly
- Credit others' contributions
- Work together toward shared goals

## Unacceptable Behavior

We do not tolerate:

- Harassment, intimidation, or discrimination
- Personal attacks or inflammatory language
- Spam, trolling, or disruptive behavior
- Sharing others' private information without consent

## Scope

This Code of Conduct applies to:

- GitHub repositories (issues, PRs, discussions)
- Community spaces (chat, forums, events)
- Public representation of the project

## Enforcement

Violations may be reported to the maintainers at: spec@promptg.io

Maintainers will:

- Review reports promptly and confidentially
- Take appropriate action (warning, temporary ban, permanent ban)
- Protect reporters from retaliation

## Attribution

This Code of Conduct is adapted from community best practices and open source standards.
